## üöÄ Getting Started

### Prerequisites

- C++ compiler (GCC, Clang, or MSVC)
- Basic understanding of C++ syntax

### Compilation

Most programs can be compiled with:

```bash
g++ -std=c++17 filename.cpp -o output
```

### Running

```bash
./output
```
